# The Sugar Cube Lemonade — Official Website

Official website for **The Sugar Cube Lemonade**, a mobile lemonade stand based in Leitchfield, Kentucky. Built as a static single-page site deployable via GitHub Pages with no build process or dependencies.

---

## Live Site

Hosted via GitHub Pages at the repository's configured Pages URL.

## Tech Stack

- Plain HTML5, CSS3, vanilla JavaScript — no frameworks, no build tools
- Google Fonts: Bangers, Permanent Marker, Special Elite, Nunito
- Scroll reveal via `IntersectionObserver`
- Fully static — deployable by dropping files into any web host or GitHub Pages repo

---

## Deployment (GitHub Pages)

1. Push all files to the `main` branch of your GitHub repository
2. Go to **Settings → Pages**
3. Set source to `Deploy from a branch`, select `main`, root `/`
4. GitHub will publish the site at `https://<your-username>.github.io/<repo-name>/`

No build step required.
